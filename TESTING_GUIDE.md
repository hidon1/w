# מדריך מהיר - בדיקת חיבור Firebase (Quick Start - Firebase Connection Testing)

## 📋 בדיקת החיבור (Connection Testing)

### שלב 1: הרץ את הסקריפט לוודא שהכל תקין
```bash
./verify-firebase.sh
```

אם כל הבדיקות עוברות (✅), המשך לשלב הבא.

### שלב 2: פתח את האפליקציה בדפדפן
```bash
# אם יש לך Python:
python3 -m http.server 8000

# לחלופין, אם יש לך Node.js:
npx http-server -p 8000

# או השתמש בכל שרת HTTP אחר
```

פתח דפדפן ולך ל: `http://localhost:8000`

### שלב 3: בדוק את באנר ההתחברות

כאשר אתה פותח את הדף לראשונה:
1. ⏱️ אחרי שנייה אחת, אמור להופיע **באנר התחברות**
2. 🔍 הבאנר כולל:
   - כפתור "התחבר עם Google"
   - טופס התחברות עם אימייל/סיסמה
   - אפשרות להירשם

### שלב 4: התחבר למערכת

#### אפשרות א': התחברות עם Google
1. לחץ על "התחבר עם Google"
2. בחר חשבון Google
3. אשר את ההרשאות
4. ✅ אמור להתחבר אוטומטית

#### אפשרות ב': הרשמה עם אימייל
1. לחץ על "הירשם כאן"
2. הזן אימייל וסיסמה (לפחות 6 תווים)
3. לחץ "הירשם"
4. ✅ אמור להירשם ולהתחבר אוטומטית

#### אפשרות ג': התחברות עם אימייל קיים
1. הזן אימייל וסיסמה
2. לחץ "התחבר"
3. ✅ אמור להתחבר אוטומטית

### שלב 5: וודא שהחיבור עובד

אחרי התחברות, בדוק:
1. 🟢 **סטטוס החיבור** בתחתית התפריט הצדדי צריך להראות "מחובר ל-Firebase" בירוק
2. 📧 **כתובת האימייל שלך** צריכה להופיע בתפריט הצדדי
3. 🔓 **כפתור התנתקות** צריך להיות זמין

### שלב 6: בדוק שמירת נתונים

1. **צור פרויקט חדש**:
   - פתח את התפריט הצדדי
   - לחץ על "ניהול פרויקטים"
   - צור פרויקט חדש
   - ✅ הפרויקט צריך להישמר אוטומטית ל-Firebase

2. **ערוך פרויקט**:
   - פתח פרויקט קיים
   - הזן נתונים (שם יין, שנת בציר, וכו')
   - ✅ כל שינוי צריך להישמר אוטומטית ל-Firebase (אחרי חצי שנייה)

3. **בדוק שהנתונים נשמרים**:
   - התנתק מהמערכת
   - התחבר מחדש
   - ✅ כל הפרויקטים והנתונים צריכים להיטען מ-Firebase

### שלב 7: בדוק ב-Firebase Console

1. פתח את [Firebase Console](https://console.firebase.google.com/project/hidon1-e4c91)
2. עבור ל-**Firestore Database**
3. צפה בנתונים תחת:
   ```
   accounts/
     {your-user-id}/
       data/
         projects/
         reminders/
         inbox/
         preferences/
   ```
4. ✅ הנתונים שלך צריכים להופיע שם

## 🐛 פתרון בעיות (Troubleshooting)

### הבאנר לא מופיע
- פתח את ה-**Developer Console** (F12)
- חפש שגיאות ב-**Console**
- בדוק אם יש שגיאות Firebase

### ההתחברות נכשלת
**Google Sign-In:**
- ודא שה-domain מאושר ב-Firebase Console
- בדוק שאין חוסם pop-ups בדפדפן

**Email/Password:**
- ודא שהאימייל תקין
- ודא שהסיסמה לפחות 6 תווים

### הנתונים לא נשמרים
1. בדוק ב-Console אם יש שגיאות
2. ודא שאתה מחובר (בדוק את סטטוס החיבור)
3. חכה חצי שנייה אחרי כל שינוי (debounce)

### הנתונים לא נטענים
1. ודא שיש לך נתונים ב-Firebase Console
2. בדוק שאתה מחובר עם אותו משתמש
3. רענן את הדף (F5)
