# סיכום שינויים - אינטגרציה מלאה עם Firebase

## מה הוסף?

הוסף מערכת אינטגרציה מלאה עם Firebase בצד לקוח, הכוללת:

### 1. אימות משתמשים (Authentication)
- **התחברות עם אימייל וסיסמה** - הרשמה והתחברות רגילה
- **התחברות עם Google** - כפתור "התחבר עם Google" בדיאלוג ההתחברות
- **ניהול מצב אימות** - עדכון אוטומטי של ממשק המשתמש
- **התנתקות** - כפתור יציאה

### 2. מסד נתונים (Firestore)

#### קולקציית accounts
כל חשבון מכיל:
- `ownerUid` - מזהה יחודי של בעל החשבון
- `members` - מערך של כתובות אימייל של כל חברי החשבון
- `createdAt` - תאריך יצירה
- `displayName` - שם תצוגה

**מזהה חשבון אנונימי:**
- פורמט: `acc_[16 תווים]`
- המזהה לא חושף מידע על הבעלים

#### תת-קולקציית messages
כל הודעה מכילה:
- `text` - תוכן טקסטואלי (או null)
- `imageUrl` - קישור לתמונה (או null)
- `sender` - אימייל השולח
- `createdAt` - תאריך שליחה

### 3. אחסון קבצים (Storage)
- **העלאת תמונות** - תמיכה בהעלאת תמונות להודעות
- **נתיב אחסון:** `messages/{accountId}/{timestamp}_{filename}`
- **הגבלות:**
  - גודל מקסימלי: 5MB
  - סוג קובץ: תמונות בלבד

### 4. ממשק משתמש

#### דיאלוג התחברות (עודכן)
- כפתור "התחבר עם Google" בכל מצב (הרשמה/התחברות)
- עיצוב כחול Google רשמי
- אייקון Google

#### פאנל ניהול Firebase (חדש)
מיקום: צד שמאל-עליון (מופעל דרך תפריט)

**תכונות:**
1. **בחירת חשבון** - תפריט נפתח להחלפה בין חשבונות
2. **ניהול משתמשים:**
   - שדה הזנת אימייל
   - כפתור "הוסף משתמש"
   - רשימת משתמשים נוכחיים
3. **שליחת הודעה:**
   - שדה טקסט חופשי
   - בחירת קובץ תמונה
   - כפתור "שלח"
4. **רשימת הודעות:**
   - עדכון אוטומטי בזמן אמת
   - הצגת טקסט ותמונה
   - חותמת זמן

#### תפריט צד (עודכן)
- אפשרות חדשה: "הודעות Firebase"

### 5. תיעוד מקיף

**קובץ FIREBASE_SETUP.md מכיל:**
- הוראות שלב-אחר-שלב להגדרת Firebase Console
- כללי אבטחה מלאים ל-Firestore
- כללי אבטחה מלאים ל-Storage
- הסבר על מבנה מסד הנתונים
- מדריך פתרון בעיות
- דוגמאות שימוש

## כיצד להשתמש?

### התחברות ראשונית

1. לחץ על "התחבר" בסרגל הצד
2. בחר אחת מהאפשרויות:
   - הרשם עם אימייל וסיסמה
   - התחבר עם Google (מומלץ - מהיר ופשוט)

### פתיחת פאנל Firebase

1. לאחר התחברות, פתח את התפריט (☰)
2. לחץ על "הודעות Firebase"
3. הפאנל יופיע בצד שמאל-עליון

### שליחת הודעה

1. כתוב טקסט בשדה "הקלד הודעה..." (אופציונלי)
2. בחר תמונה (אופציונלי)
3. לחץ "שלח"
4. ההודעה תופיע מיד ברשימה

### הוספת משתמש לחשבון

**שים לב:** רק בעל החשבון יכול להוסיף משתמשים

1. הזן כתובת אימייל של המשתמש
2. לחץ "הוסף משתמש"
3. המשתמש יראה את החשבון ברגע שיתחבר

### מעבר בין חשבונות

אם את/ה חבר במספר חשבונות:
1. השתמש בתפריט הנפתח "בחר חשבון"
2. בחר את החשבון הרצוי
3. ההודעות והמשתמשים יתעדכנו אוטומטית

## כללי אבטחה - סיכום מהיר

### Firestore
```
- רק חברי חשבון רואים את הנתונים שלו
- רק בעל החשבון יכול להוסיף/להסיר משתמשים
- הודעות הן immutable (לא ניתן לערוך/למחוק)
```

### Storage
```
- רק חברי חשבון יכולים להעלות ולראות תמונות
- גודל מקסימלי: 5MB
- סוג קובץ: תמונות בלבד
```
