## 📖 איך להתחיל?

### שלב 1: בדיקה אוטומטית
```bash
./verify-firebase.sh
```
אם כל הבדיקות עוברות (✅) - הכל תקין!

### שלב 2: הפעלת השרת
```bash
python3 -m http.server 8000
```

### שלב 3: פתח בדפדפן
```
http://localhost:8000
```

### שלב 4: התחבר!
- אחרי שנייה יופיע באנר התחברות 🎉
- בחר: Google / אימייל-סיסמה
- זהו! כל הפרויקטים שלך יישמרו ל-Firebase

## 📊 מבנה הנתונים ב-Firebase

```
accounts/
  {your-user-id}/
    data/
      projects/      ← כל הפרויקטים שלך
        - projects: {}
        - stagesConfig: {}
        - updatedAt: timestamp
      reminders/     ← תזכורות
      inbox/         ← תיבת דואר
      preferences/   ← העדפות
```

## 💡 טיפים

1. **אין באנר התחברות?** 
   - בדוק אם `localStorage.getItem('skipAuthModal')` מוגדר
   - רענן את הדף

2. **נתונים לא נשמרים?**
   - ודא שאתה מחובר (בדוק סטטוס חיבור)
   - חכה חצי שנייה אחרי כל שינוי

3. **רוצה לעבוד בלי חיבור?**
   - פשוט אל תתחבר - המערכת תעבוד מקומית
