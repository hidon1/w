### ✅ 4. שיתוף גישה למשתמשים (לשונית 1)

**תכונות:**
- 📧 הוסף משתמשים לפי כתובת מייל
- 🔐 הגדר הרשאות: צפייה בלבד או עריכה
- 💰 בחר אם המשתמש יכול לראות מידע כספי
- 📁 הענק גישה לפרויקט ספציפי או לכל הפרויקטים
- 👥 צפה ברשימת משתמשים פעילים
- 🗑️ בטל גישה בקליק אחד

**איך זה עובד:**
1. משתמש A נכנס לחשבון שלו
2. הוא פותח "ניהול גישה" → "שיתוף גישה למשתמשים"
3. הוא מזין את המייל של משתמש B
4. בוחר הרשאות (צפייה/עריכה + גישה כספית)
5. משתמש B יכול להיכנס עם המייל שלו ולראות את הפרויקט של A!
6. כל שינוי שמשתמש A עושה - משתמש B רואה בזמן אמת ולהיפך!

---

### ✅ 5. חיבור כתובות מייל (לשונית 2)

**תכונות:**
- 🔗 חבר מספר כתובות מייל לאותו חשבון
- 🌍 התחבר ממכשירים שונים עם מיילים שונים
- 🆔 כל המיילים משתמשים באותו UID (מזהה ייחודי)
- 👀 צפה בכל ספקי האימות המקושרים (Google, Email/Password)

**איך זה עובד:**
1. אתה מחובר עם Google (example@gmail.com)
2. אתה רוצה גם להתחבר מהעבודה (work@company.com)
3. פותח "ניהול גישה" → "חיבור כתובות מייל"
4. מזין את work@company.com וסיסמה חדשה
5. לוחץ "שייך כתובת מייל חדשה"
6. עכשיו תוכל להתחבר עם שני המיילים לאותו חשבון!

### ✅ 6. סנכרון בזמן אמת

**מה זה אומר:**
- כשמשתמש אחד משנה משהו, כל המשתמשים האחרים רואים את השינוי **מיד**
- לא צריך לרענן את הדף
- עובד גם במספר מכשירים של אותו משתמש
- השינויים נשמרים אוטומטית ל-Firebase

**איך זה עובד:**
- Firebase Firestore Persistence מופעל
- כל שינוי בנתונים מעדכן אוטומטית את כל המשתמשים המחוברים
- אפילו אם אין אינטרנט - השינויים נשמרים מקומית ומסתנכרנים כשהאינטרנט חוזר

## הוראות שימוש

### איך לשתף פרויקט עם משתמש אחר:

1. **פתח את התפריט** (המבורגר בצד ימין למעלה)
2. **לחץ על "ניהול גישה"**
3. **בחר בלשונית "שיתוף גישה למשתמשים"** (ברירת מחדל)
4. **הזן את כתובת המייל** של המשתמש שרוצה לשתף איתו
5. **בחר פרויקט** (ספציפי או כל הפרויקטים)
6. **בחר הרשאות:**
   - **צפייה בלבד** - המשתמש רק רואה, לא יכול לערוך
   - **עריכה** - המשתמש יכול גם לשנות ולהוסיף
7. **סמן "גישה לניהול כספים"** אם רוצה שהמשתמש יראה מידע כספי
8. **לחץ "הענק גישה"**
9. ✅ **סיימת!** המשתמש השני יכול להתחבר ולראות את הפרויקט שלך

### איך לחבר מייל נוסף לחשבון שלך:

1. **פתח "ניהול גישה"**
2. **עבור ללשונית "חיבור כתובות מייל"**
3. **הזן כתובת מייל חדשה**
4. **הזן סיסמה** (לפחות 6 תווים)
5. **לחץ "שייך כתובת מייל חדשה"**
6. ✅ **סיימת!** עכשיו תוכל להתחבר עם המייל החדש

## פריסה (Deployment)

כדי להעלות את כל השינויים ל-Firebase:

```bash
# התחבר ל-Firebase (אם עדיין לא)
firebase login

# פרוס הכל
firebase deploy

# או פרוס חלקים ספציפיים:
firebase deploy --only hosting        # רק האתר
firebase deploy --only firestore:rules # רק חוקי Firestore
firebase deploy --only storage:rules   # רק חוקי אחסון
```
